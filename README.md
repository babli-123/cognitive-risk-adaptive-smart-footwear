Cognitive Risk Adaptive Smart Footwear

Project Overview

This repository contains the documentation for our **group academic final-year project**, which is currently under development as part of the B.E. Computer Science and Engineering (Data Science) program.

The project aims to improve worker safety using Artificial Intelligence (AI), Machine Learning, TinyML, and IoT technologies. It is designed to monitor workers in real time, detect potential risks such as fatigue and abnormal movements, and provide early safety alerts.

Objectives

Improve worker safety through intelligent monitoring.
Detect fatigue and abnormal movements.
Provide real-time safety alerts.
Reduce workplace accidents.
Apply AI, TinyML, and IoT technologies for smart risk detection.

Technologies Used

Python
Machine Learning
TinyML
IoT

Sensors
